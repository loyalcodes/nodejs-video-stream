## Description
A NodeJS video streaming service powered by NestJS

## Installation

```bash
#clone repostory
$ git clone https://github.com/loyalcodes/video-streaming-service.git
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
