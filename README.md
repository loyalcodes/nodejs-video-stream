## Description

[Video Streaming Service](https://github.com/nestjs/nest) A NodeJS video streaming service powered by NestJS

## Installation

```bash
# install dependencies
$ npm 
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
